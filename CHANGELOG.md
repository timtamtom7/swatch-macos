# Changelog

All notable changes to Swatch will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-25

### Added
- Menu bar color picker with eyedropper tool
- Multiple color format support (HEX, RGB, HSB, CMYK, Swift, NSColor)
- Color history tracking (last 20 colors)
- Palette management (create, edit, delete palettes)
- Import/export palettes (JSON, CSS, Swift, SwiftUI, ASE formats)
- Color harmony generation (complementary, triadic, analogous, split complementary, tetradic)
- WCAG accessibility contrast checker (AA/AAA levels)
- Color blindness simulation (protanopia, deuteranopia, tritanopia)
- Image color extraction
- Widget extension with Color of the Day and Palette widgets
- Notification support (daily color reminders, weekly palette digests)
- Full VoiceOver and accessibility support
- Keyboard navigation
- Dynamic Type support
- Reduced Motion support
- macOS 13+ native application
